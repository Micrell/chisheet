# Rename file
```sh
git mv <file>
```

# reset

Remove file added files
```sh
git reset <file>
```

# ENV var

```sh
$env:GIT_SSH_COMMAND = "ssh -p <port> -i ""C:\Users\<user>\.ssh\<key>"""
```


# log

| OPTION    | EFFECT                      |

| --reverse | show history from beginning |

Show git history
```
git log
```
